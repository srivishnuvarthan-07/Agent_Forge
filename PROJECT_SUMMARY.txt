WHAT IS AGENTFORGE?
-------------------
AgentForge is a Low-Code Multi-Agent Builder that lets users visually design and
run hierarchical AI agent workflows on a drag-and-drop canvas. It uses CrewAI for
agent orchestration, Groq LLM (llama-3.3-70b-versatile) as the language model,
ChromaDB for shared agent memory, and FastAPI + WebSockets for the backend API.

Users place agent nodes (CEO, Researcher, Analyst, Developer, etc.) on a canvas,
connect them with edges to define hierarchy and collaboration, then submit the
canvas to the backend which runs the crew and streams real-time events back over
WebSocket.

CANONICAL SERVER COMMAND
------------------------
    uvicorn api.main:app --reload

TEST COMMAND
------------
    python -m pytest tests/ -v
